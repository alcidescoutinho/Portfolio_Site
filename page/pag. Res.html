Quem sou eu?

Sou um cientista de dados de Maceió — Alagoas, formado em engenharia de petróleo pela UNIT-AL. Meu contado com a ciência de dados começou com o meu TCC, no qual criei um modelo de inteligência coletiva para calcular o melhor modelo de um design de poço de petróleo, desde então. Ao final da vaga comecei a cursar uma pós-graduação de gestão de projetos e em paralelo comecei a atuar de forma autônoma no mercado financeiro, no qual fiquei 3 anos trabalhando, como sempre utilizei meus conhecimentos de engenheiro para estudar o mercado de ações, comecei a programar em python e conheci então a ciência de dados que desde o início foi amor a "segunda vista?" comecei a migrar e estudar essa área e hoje busco atuar como um profissional da área. Como cientista de dados júnior, fora as dezenas de bancos de dados trabalhados online, a minha primeira experiência real com prestação de serviços com dados foi para empresa do ramo agrícola, onde fui responsável por elaborar uma análise dos dados com power BI e elaborar alguns modelos de machine learning para a previsão de produção por toneladas.

Meus Conhecimentos e experiências

Como citado anteriormente, minha experiência se consiste bem na utilização do python e suas diversas bibliotecas para ciência de dados, como pandas, numpy, sklearn, tensorflow, matplotlib, seaborn, etc. Com o power BI também tenho experiência, principalmente a correlação entre python e power BI ou Power BI e MYSQL facilita muito o trabalho no desenvolvimento de relatórios. Com a linguagem SQL tenho conhecimentos com o MYSQL, ferramenta que tenho bastante familiaridade e facilidade.

Sobre Conhecimentos de machine learning, como visto no meu portfólio, tenho bastante costume e familiaridade com suas diversas aplicações, seja para modelo de regressão/classificação como clusterização, deep learning e NLP. Sobre cursos das ferramentas, tenho tanto de SQL como de Power BI e atualmente estou cursando o Data Analystic da Google, buscando logo em seguida adquirir conhecimento sobre spark, a depender de quando você esteja lendo, talvez já tenha concluído e esquecido de atualizar essas informações. Outros conhecimentos como GitHub e pacote Office sempre me mantenho atualizado e sem problemas de uso, além também de ainda manter algum conhecimento dos métodos ágeis.

Sobre línguas estrangeiras, atualmente meu nível de inglês é considerado avançado nos três aspectos de comunicação: fala, escrita e audição. Continuo estudando para me manter atualizado e com a fluência constante (escrevendo trabalhos no kaggle em inglês como exemplo).
